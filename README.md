# Cypher Studio Portfolio

A premium app showcase portfolio built with Vue 3, Vite, TypeScript, and Tailwind CSS.

## Features
- **Modern Design**: Glassmorphism, smooth gradients, and subtle animations.
- **Project Gallery**: Grid of apps with unique branding and detail pages.
- **Showcase Pages**: Deep dives into each app with key features, stats, and live demos.
- **Responsive**: Mobile-first design optimized for all screen sizes.
- **Performance**: Built with Vite for ultra-fast development and production builds.

## Tech Stack
- **Framework**: Vue 3 (Composition API)
- **Styling**: Tailwind CSS
- **Routing**: Vue Router
- **Icons**: Lucide Vue Next

## Getting Started

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Run dev server**:
   ```bash
   npm run dev
   ```

3. **Build for production**:
   ```bash
   npm run build
   ```

## Development
- `src/data/projects.ts`: Manage your app projects here.
- `src/views/HomeView.vue`: Main landing page components.
- `src/views/AppDetailView.vue`: Reusable app showcase template.
- `src/style.css`: Global design tokens and glassmorphism styles.
